1
Curl — ∇ × F
What does curl mean?

Curl measures how much a vector field rotates or swirls around a point. Think of it like placing a tiny paddle wheel in a river — if it spins, there's curl. If it doesn't spin, curl is zero.

  • Curl produces a vector (in 3D)
  • In 2D problems, we only care about the z-component of curl
  • Curl = 0 → the field is called irrotational
Formula — 3D Curl

For F = <P, Q, R> or F = Pi + Qj + Rk:

The determinant method (easiest to remember)
curl F = ∇ × F = | i j k |
            | ∂/∂x ∂/∂y ∂/∂z |
            | P Q R |
Expanded result
curl F = (R_y − Q_z)i − (R_x − P_z)j + (Q_x − P_y)k
Memory trick: For each component, subtract the mixed partial derivatives in a cyclic pattern: (yz), (xz), (xy). The j-component gets a minus sign.
Formula — 2D Curl (most common in exams)

For F = <P(x,y), Q(x,y)> (R = 0):

curl F = ∂Q/∂x − ∂P/∂y   ← this is the z-component only
When is curl = 0? If ∂Q/∂x = ∂P/∂y everywhere, the field is conservative (irrotational). This also means the line integral is path-independent!
⚡ Solved Example 1.1 — 2D Curl

Find the curl of F = <x²y, xy² + x>

1
Identify P and Q
P = x²y    Q = xy² + x
2
Compute partial derivatives
∂Q/∂x = y² + 1     ∂P/∂y = x²
3
Apply the formula
curl F = ∂Q/∂x − ∂P/∂y = (y² + 1) − x²
✓
Answer
curl F = y² − x² + 1

Since this is not zero, the field is not conservative.

⚡ Solved Example 1.2 — 3D Curl

Find curl F for F = <yz, xz, xy>

1
Set up: P = yz, Q = xz, R = xy
2
i-component: R_y − Q_z
∂(xy)/∂y − ∂(xz)/∂z = x − x = 0
3
j-component: −(R_x − P_z)
−[∂(xy)/∂x − ∂(yz)/∂z] = −[y − y] = 0
4
k-component: Q_x − P_y
∂(xz)/∂x − ∂(yz)/∂y = z − z = 0
✓
Answer
curl F = <0, 0, 0>

Curl is zero → F is conservative. A potential function exists.

2
Divergence — ∇ · F
What does divergence mean?

Divergence measures how much a vector field spreads out from or converges to a point. Think of sources (water sprinklers) and sinks (drains).

div F > 0 Field is spreading out → source
div F < 0 Field is converging → sink

Divergence = 0 → field is called incompressible or solenoidal

Formula — Divergence

Divergence produces a scalar (just a number, not a vector!)

3D: F = <P, Q, R>
div F = ∇ · F = ∂P/∂x + ∂Q/∂y + ∂R/∂z
2D: F = <P, Q>
div F = ∇ · F = ∂P/∂x + ∂Q/∂y
Key difference from curl: Divergence is always a scalar. Curl (in 3D) is a vector. In 2D, curl gives a scalar (the z-component only). Don't mix these up!
⚡ Solved Example 2.1 — 2D Divergence

Find div F for F = <x³ + y², 3xy − y³>

1
Identify P and Q
P = x³ + y²    Q = 3xy − y³
2
Compute partial derivatives
∂P/∂x = 3x²     ∂Q/∂y = 3x − 3y²
3
Add them
div F = 3x² + 3x − 3y²
✓
Answer
div F = 3x² + 3x − 3y²
⚡ Solved Example 2.2 — 3D Divergence

Find div F for F = <x²z, y sin(z), ze^(xy)>

1
∂P/∂x (differentiate x²z with respect to x)
∂(x²z)/∂x = 2xz
2
∂Q/∂y (differentiate y sin(z) with respect to y)
∂(y sin z)/∂y = sin(z)
3
∂R/∂z (differentiate ze^(xy) with respect to z)
∂(ze^(xy))/∂z = e^(xy)
✓
Answer
div F = 2xz + sin(z) + e^(xy)
Quick Comparison: Curl vs Divergence
Property Curl (∇ × F) Divergence (∇ · F)
Output type Vector (in 3D), scalar (in 2D) Scalar always
Measures Rotation / swirling Expansion / compression
= 0 means Irrotational (conservative) Incompressible (solenoidal)
Operation Cross product: ∇ × F Dot product: ∇ · F
2D formula ∂Q/∂x − ∂P/∂y ∂P/∂x + ∂Q/∂y
3
Line Integrals — ∫_C F · dr
What is a line integral?

A line integral computes the total effect of a vector field along a curve. Physical meaning: work done by force F moving along path C.

The key idea: You parametrize the curve C as r(t) = (x(t), y(t)), then convert everything into a single variable t and integrate normally.
Step-by-Step Method
Line integral formula
∫_C F · dr = ∫_a^b F(r(t)) · r'(t) dt
Expanded (F = <P,Q>, r = <x,y>)
= ∫_a^b [ P(x,y) dx/dt + Q(x,y) dy/dt ] dt

Always follow these 4 steps:

1
Parametrize C — write x(t), y(t) and find a, b
2
Compute dr/dt — find dx/dt and dy/dt
3
Substitute — replace x, y in F with x(t), y(t)
4
Integrate — compute the definite integral in t
Common Parametrizations (Memorize These!)
Curve Type Parametrization Range
Straight line from (x₁,y₁) to (x₂,y₂) x = x₁ + t(x₂−x₁)
y = y₁ + t(y₂−y₁)		 t ∈ [0, 1]
Circle x²+y²=r², counterclockwise x = r cos(t)
y = r sin(t)		 t ∈ [0, 2π]
y = f(x), left to right x = t
y = f(t)		 t ∈ [a, b]
x = g(y), bottom to top x = g(t)
y = t		 t ∈ [a, b]
⚡ Solved Example 3.1 — Line integral along a line segment

Evaluate ∫_C (xy) dx + (x + y) dy where C is the line segment from (0,0) to (1,2).

1
Parametrize
x = t,   y = 2t,   t ∈ [0, 1]
(Use the standard line segment formula with (x₁,y₁)=(0,0), (x₂,y₂)=(1,2))
2
Find dx/dt and dy/dt
dx/dt = 1,    dy/dt = 2
3
Substitute P = xy and Q = x + y
P = xy = (t)(2t) = 2t²
Q = x + y = t + 2t = 3t
4
Build the integral
∫₀¹ [2t²(1) + 3t(2)] dt = ∫₀¹ [2t² + 6t] dt
5
Integrate
= [2t³/3 + 3t²]₀¹ = 2/3 + 3 = 2/3 + 9/3
✓
Answer
= 11/3
⚡ Solved Example 3.2 — Line integral around a circle

Evaluate ∫_C −y dx + x dy where C is the unit circle x²+y²=1 (counterclockwise).

1
Parametrize the circle
x = cos t,   y = sin t,   t ∈ [0, 2π]
2
Derivatives
dx/dt = −sin t,    dy/dt = cos t
3
Substitute P = −y = −sin t, Q = x = cos t
∫₀²π [(−sin t)(−sin t) + (cos t)(cos t)] dt
= ∫₀²π [sin²t + cos²t] dt = ∫₀²π 1 dt
✓
Answer
= 2π

Note: This equals 2 × (area of circle) = 2π(1)². This is actually Green's Theorem in action!

Shortcut: Conservative Fields

If F is conservative (curl = 0), then there exists a potential function f such that F = ∇f, and:

∫_C F · dr = f(end point) − f(start point)
Huge time saver! First check if curl F = 0. If yes, find potential function f (by integration), then just plug in endpoints. No need to parametrize at all!
4
Green's Theorem ∮ = ∬
The Big Idea

Green's Theorem connects a line integral around a closed curve to a double integral over the enclosed region. It's a shortcut — instead of integrating around the boundary, integrate over the area inside.

When to use it: The curve C must be closed (starts and ends at same point), positively oriented (counterclockwise), and piecewise smooth.
The Formula
Green's Theorem
∮_C P dx + Q dy = ∬_D (∂Q/∂x − ∂P/∂y) dA
Left side Line integral around closed curve C
Right side Double integral over region D inside C
Notice: The integrand on the right side is exactly the 2D curl (∂Q/∂x − ∂P/∂y)! Green's Theorem = "curl integrated over area = circulation around boundary"
Strategy: Which Direction to Convert?
SituationConvert toWhy
Curve C is complicated, region D is simple → Double integral Easier to integrate over the region
Region D is complicated, C is simple → Line integral Easier to walk along the boundary
∂Q/∂x − ∂P/∂y = constant → Double integral Just multiply by area!
Finding area of region → Use area formula Area = ½ ∮ (x dy − y dx)
Bonus: Area Using Green's Theorem
Area of region D enclosed by C
A = ½ ∮_C (x dy − y dx) = ∮_C x dy = −∮_C y dx

These all give the same answer — use whichever is easiest to integrate.

⚡ Solved Example 4.1 — Applying Green's Theorem (forward)

Evaluate ∮_C (y³) dx + (x³) dy where C is the circle x²+y²=4 (counterclockwise).

1
Identify P and Q
P = y³    Q = x³
2
Compute the curl integrand
∂Q/∂x − ∂P/∂y = 3x² − 3y²
3
Apply Green's Theorem → switch to double integral over disk x²+y²≤4
= ∬_D (3x² − 3y²) dA
4
Convert to polar: x = r cosθ, y = r sinθ, dA = r dr dθ, r ∈ [0,2], θ ∈ [0,2π]
3x² − 3y² = 3r²cos²θ − 3r²sin²θ = 3r² cos(2θ)
∫₀²π ∫₀² 3r² cos(2θ) · r dr dθ = ∫₀²π cos(2θ) dθ · ∫₀² 3r³ dr
5
Evaluate the θ-integral
∫₀²π cos(2θ) dθ = [sin(2θ)/2]₀²π = 0
✓
Answer
= 0

The θ-integral being zero kills the whole thing. Without Green's Theorem, parametrizing this circle and cubing would be a nightmare!

⚡ Solved Example 4.2 — Constant integrand (fastest case)

Evaluate ∮_C (2y) dx + (5x) dy where C is the square with vertices (0,0),(2,0),(2,2),(0,2) counterclockwise.

1
Identify P and Q
P = 2y    Q = 5x
2
Compute the curl integrand
∂Q/∂x − ∂P/∂y = 5 − 2 = 3
⟵ constant! This is the easiest case.
3
Apply Green's Theorem
∬_D 3 dA = 3 × (Area of D)
4
Area of square = 2 × 2 = 4
3 × 4 = 12
✓
Answer
= 12

No integration needed! When the curl integrand is a constant, just multiply by the area.

★
Exam Cheatsheet — Common Mistakes & Fixes
❌ Common Mistakes
  • Forgetting the minus sign in the j-component of 3D curl
  • Confusing curl (vector) with divergence (scalar)
  • Using the wrong parametrization range
  • Forgetting Green's Theorem needs counterclockwise orientation
  • Not substituting x(t), y(t) into F before integrating
  • Mixing up ∂Q/∂x and ∂P/∂y order in curl formula
✅ Quick Checks
  • Curl 2D = ∂Q/∂x − ∂P/∂y (Q first!)
  • Divergence 2D = ∂P/∂x + ∂Q/∂y (add, don't subtract)
  • Always parametrize before computing a line integral
  • If curl = 0 on a line integral → use the potential function shortcut
  • Green's Theorem: only works for closed curves
  • If ∂Q/∂x − ∂P/∂y = constant → area trick!
Decision Tree: Which Method to Use?
Given a line integral ∮_C P dx + Q dy:
  → Is C closed? → Try Green's Theorem
     → Is ∂Q/∂x − ∂P/∂y = constant? → Multiply by area — done!
     → Otherwise → set up double integral in polar or Cartesian
  → Is C not closed?
     → Is F conservative (curl = 0)? → Find potential fn, plug in endpoints
     → Otherwise → Parametrize C, substitute, integrate in t
